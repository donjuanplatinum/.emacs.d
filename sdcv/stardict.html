2 Emacs Plugin

  • Install sdcv(use your system package manager like apt-get/yum or download from http://sdcv.sourceforge.net/).

  • Download sdcv-mode.el from https://code.google.com/p/sdcv-mode/, put it into the emacs load path, then put these lines into you .emacs file: 

    (require 'sdcv-mode)
(global-set-key (kbd "C-c d") 'sdcv-search)
  • Lookup word use C-c d
